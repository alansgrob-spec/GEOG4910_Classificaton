# Juniper Treatment GIS Project

## Overview
This repository contains a GIS project created in ArcGIS Pro to support
juniper treatment planning. The project includes terrain slope analysis
and property boundary data.

## Folder Structure
- **data/**  
  Folder included to demonstrate standard GIS repository organization.

- **code/**  
  Contains notes documenting the workflow used in ArcGIS Pro.

## Data Organization Note
Spatial datasets are stored alongside the ArcGIS Pro project file to
preserve data paths and ensure the project opens without broken links.
The `data/` folder is included to meet course organization requirements.

## Software
- ArcGIS Pro

## How to Use
Download the repository and open `juniper treatment.aprx` in ArcGIS Pro
without changing the folder structure.
